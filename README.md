<h1 align="center" style="margin: 30px 0 30px; font-weight: bold;">灵犀</h1>
<h4 align="center">基于RuoYi（SpringBoot+Vue前后端分离）的Java快速开发框架</h4>


## 背景介绍

灵犀智能零售终端管理系统是基于 RuoYi 框架开发的一款前后端分离的智能零售管理平台，采用 SpringBoot + Vue 3 + Element Plus + Vite 技术栈。该系统旨在提升传统售货机的运营、运维效率，通过运营数据的采集和分析不断优化运营方案，降低运营、运维成本，缩短B端用户的盈利周期。

系统通过物联网技术实现对售货机设备的远程监控和管理，支持多种业务模式，包括自营模式、加盟模式和点位主分成模式等。针对不同的摆放点位及商业场景，可匹配不同机型及不同的商品，供B端用户贴合自身特点来快速实现商业价值。

## 业务介绍

灵犀智能零售终端管理系统主要包含以下核心业务模块：

### 1. 平台管理端
平台管理端是系统的核心管理模块，主要功能包括：
- **设备管理**：售货机设备的增删改查、状态监控、货道配置等
- **商品管理**：商品分类、商品信息、库存管理等
- **工单管理**：设备维修工单、运营工单的创建、分配、跟踪和处理
- **订单管理**：销售订单的查询、统计和分析
- **合作商管理**：合作商信息、权限管理和数据统计
- **区域管理**：点位区域的划分和管理
- **策略管理**：设备运营策略的配置和管理

### 2. 监控中心
监控中心用于实时监控系统运行状态，主要功能包括：
- **服务器监控**：系统服务器的运行状态和性能指标
- **缓存监控**：Redis缓存的使用情况和性能
- **定时任务监控**：系统定时任务的执行状态和日志
- **登录日志**：用户登录记录和异常登录监控
- **在线用户监控**：当前在线用户的管理和会话控制
- **操作日志**：系统操作的审计日志

### 3. 系统管理
系统管理用于配置和管理系统基础功能，主要包括：
- **部门管理**：企业部门结构的管理
- **菜单管理**：系统菜单的配置和权限控制
- **角色管理**：用户角色的创建、修改和权限分配
- **用户管理**：系统用户的增删改查和权限管理
- **字典管理**：系统字典数据的维护
- **配置管理**：系统参数的配置和管理
- **通知管理**：系统通知的发布和管理
- **岗位管理**：用户岗位的管理

### 4. AI功能
系统集成了AI聊天功能，支持与大语言模型进行交互，主要用于：
- 智能客服：解答用户疑问
- 数据分析：辅助运营数据分析
- 智能建议：基于数据提供运营建议

### 5. 工具管理
工具管理模块提供了开发和维护工具，主要包括：
- **代码生成**：自动生成前后端代码
- **构建工具**：可视化构建页面组件
- **Swagger文档**：API接口文档的生成和管理

### 6. 权限管理
系统采用基于角色的权限控制，根据用户角色提供不同的功能权限：
- **运维人员**：负责设备故障处理、维修工单管理等运维相关功能
- **运营人员**：负责商品管理、订单管理、运营工单等运营相关功能
- **管理员**：拥有系统全部功能权限，负责系统配置和管理

系统通过模块化设计，实现了各功能模块的解耦和独立扩展，能够满足不同规模和业务场景的需求。
